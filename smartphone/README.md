# Smartphone Project

## Description
Java application developed to practice object-oriented programming concepts such as inheritance, interfaces, polymorphism, and user interaction through a menu system.

---

## Objective
The goal of this project is to simulate the behavior of a smartphone that can:
- Make calls
- Take photos
- Activate an alarm

These functionalities are implemented using classes, inheritance, and interfaces.

---

## Exercise Statement
Create a base class called `Phone` with basic information such as brand and model.

Then create two interfaces:
- `Camera` → for taking photos
- `Clock` → for activating an alarm

Finally, create a `Smartphone` class that:
- Extends `Phone`
- Implements both `Camera` and `Clock`

---

## Features
- Class inheritance (`extends`)
- Interface implementation (`implements`)
- Method overriding
- Polymorphism
- Interactive menu using `Scanner`
